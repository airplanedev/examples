# Airplane Examples

This repo contains various examples that showcase how you can use Airplane.

To get started, sign up for an account at: [app.airplane.dev](https://app.airplane.dev).

## Deploy

You can deploy any of the examples in this repo using the Airplane CLI.

First, install the CLI using the [instructions here](https://github.com/airplanedev/cli#installation).

Then, deploy any example:

```sh
# f.e., to deploy the "Hello World" example in Node.js:
airplane tasks deploy -f github.com/airplanedev/examples/node/hello-world-javascript/airplane.yml
```
